품목분류 코드

HS CODE가 무엇인가요?
  • HS CODE (Harmonized Commodity Description and Coding System)는
    WCO 세계관세기구에서 관세징수 및 통계작성을 위해 분류한 품명 및 부호체계입니다.
  • 세계 공통은 6단위이며, 6단위 이하는 국가마다 상이하게 분류됩니다.
    한국은 국제 공통 6자리에 더해 물건의 세부 분류를 위해 4자리를 추가한 HSK 코드(10자리)를 사용하고 있습니다.
    (한국 10단위, 미국 10단위, 일본 9단위, 중국 8단위까지 분류)
  • HS CODE 분류체계
    • 1~2자리(류): 재질과 기능에 따라 전물품을 분류
    • 3~4자리(호): 가공단계별로 분류. 무역량 및 기타 상품개발에 따른 호의 균등을 유지하기 위해
      • -  분할 : 무역량이 많거나 통계집계를 필요로 한 품목
      • -  이전 : CCCN체제의 정비에 따른 동일 상품군의 형성
      • -  삭제 : 무역량이 미미한 품목
      • -  추가 : 각국의 특산물, 신개발품, 무역거래량이 많은 품목들.
    • 5~6자리(소호): 상기기준에 입각하면서 운송, 보험들에도 적용, 사용할 수 있도록 확대.
    • 7~10자리(통계부호): 각국의 경제적 사정에 따라 자발적으로 분류할 수 있는 Code.
    • 국제 무역에 있어 HS code는 첫 4자리 + 2자리가 중요한 부분이고 통일되어 있음. (예 : 3402.11- )
    • HS code는 무역 당사자 간 보다는 무역 당사국에 필요한 자료임 (수입시 세금부과를 위해, 수출품의 통제 및 통계를 위해)
HS CODE는 왜 필요한가요?
  • HS CODE는 수입 시 부과되는 세율을 결정하는 기준이며, 수출입 통제 및 통계 분류를 위해 필요합니다.
  • HS CODE를 통해 품목에 부과되는 관세를 검색하거나 수출 및 수입품의 통계를 확인할 수 있습니다.
  • 수출입신고서의 세번부호란에 HS CODE를 기재해야 합니다. 수출입신고시 HS CODE를 잘못 입력한 경우, 가산세가 부과될 수 있습니다.
  • 관세법 제226조에 따른 수출입 요건을 확인하기 위해서는 HS CODE가 필요합니다.
  • 간이정액환급액은 HS CODE에 따라 결정됩니다. 정확한 HS CODE를 알아야 관세의 과다 또는 과소환급의 위험성을 줄어듭니다.
  • K-stat에서 HS CODE를 통해 자세한 무역통계를 확인할 수 있습니다.
MTI CODE와는 무엇이 다른가요?
  • HS code는 WCO 세계관세기구가 만든 품목분류 코드입니다.
    6단위는 세계 공통이며, 한국은 4자리를 추가한 10자리의 HSK 코드를 사용합니다.
  • MTI CODE는 산업통상자원부가 무역동향 분석을 위해 만든 6단위의 품목분류 코드입니다.
    비슷한 종류의 HS 코드를 묶어 코드와 품목명을 부여한 분류 체계입니다.
